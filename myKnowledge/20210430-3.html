Java数据库编程

蒙大明

 

JDBC（Java Database Connectivity，Java数据库连接），是一种与平台无关的用于执行SQL语句的标准Java API，便于实现多种关系型数据库的统一操作。本文将记录Java与数据库间的操作，提别是MySQL数据库的连接与操作。

MySQL数据库

MySQL是一个小型关系型数据库管理系统，广泛应用在中小型网站中。

MySQL完全支持SQL语法。
详见SQL语法：SQL 教程 | 菜鸟教程 (runoob.com)

SQL(结构查询语言)是用于访问和处理数据库的标准的计算机语言,支持的数据库包括：MySQL、SQL Server、Access、Oracle、Sybase、DB2 等主流数据库。

MySQL的数据操作

 

MySQL中的数据类型

MySQL支持三种数据类型：数值、日期/时间和字符串(字符)类型。

详见：MySQL 数据类型 | 菜鸟教程 (runoob.com)

 

SQL语法

MySQL完全兼容。

增加数据

INSERT INTO 表名称[(字段1,字段2,...,字段n)] VALUES（值1,值2,...,值n);

例子：INSERT INTO user (name,password,age,sex,birthday) VALUES ('李兴华','www.mldn.cn', 30,'男','2008-08-27');

如果数据是字符串，则用“ ' ”括起来，日期的格式：yyyy-mm-dd

删除数据

DELETE FROM 表名称 [删除条件];

例子：DELETE FROM user WHERE id=1;

应当指定条件，否则会删除一张表的全部数据，条件一般为id

更新数据（修改）

UPDATE 表名称 SET 字段 1=值 1,...,字段 n=值 n [WHERE 更新条件];

例子：UPDATE user SET name='MLDN',age=5,birthday='2000-08-27' WHERE id=2

查询数据

SELECT {*|column alias} FROM 表名称 别名 [WHERE condition(s)];

例子：查询user表中的姓名和生日 SELECT name,birthday FROM user;

例子：查询姓名或密码中包含字母m的用户 SELECT * FROM user WHERE name LIKE '%m%' OR password LIKE '%m%';

%表示任意数据

 

取出多个数据：用LIMIT语句

SELECT 字段 FROM 表 WHERE 条件 LIMIT 开始行,取出的数据个数;

例子：取出第1~5行的记录 SELECT * FROM user WHERE name LIKE '%m%' OR password LIKE '%m%' LIMIT 0,5;
